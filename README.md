# sao-mvcaspnetcore

> my mind-blowing SAO generator

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao mvcaspnetcore my-project
```

### From git

```bash
sao margathorp/sao-mvcaspnetcore my-project
```
